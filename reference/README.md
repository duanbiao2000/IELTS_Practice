# Reference Materials

This directory contains reference materials for IELTS preparation, including:

- Official IELTS practice materials
- Sample tests and past papers
- Preparation guides and study resources
- Official IELTS documentation and specifications

## Structure

- `/reference/` - All reference materials organized by topic
- `/reference/past-papers/` - Previous IELTS test papers and answers
- `/reference/guides/` - Official IELTS preparation guides
- `/reference/specifications/` - IELTS test specifications and band descriptors

## Usage

Reference materials should be reviewed before practice sessions to understand:
- Test format and requirements
- Scoring criteria and band descriptors
- Common question types and patterns
- Official preparation recommendations

> **Note**: Reference materials are read-only study resources for understanding the IELTS format and standards.
