# Study Notes

This directory contains personal study notes, key insights, and strategic approaches for IELTS preparation.

## Structure

- `grammar/` - Grammar rules, examples, and common patterns
- `reading-techniques/` - Reading comprehension strategies and tips
- `writing-structures/` - Writing frameworks, templates, and structures

## Usage

Study notes should be actively maintained and updated as you learn:

1. **Add New Concepts** - Document new grammar rules and vocabulary
2. **Update Techniques** - Refine reading and writing strategies
3. **Track Patterns** - Note common question types and best approaches
4. **Review Regularly** - Periodically review notes to reinforce learning

## Organization Tips

- Use clear headings and subheadings
- Include examples for each concept
- Highlight key points and common pitfalls
- Cross-reference related topics
- Keep notes concise but comprehensive

## Reference Integration

Cross-reference with reference materials:
- Link to specific reference documents
- Note official guidelines vs. personal insights
- Include page numbers or section references for key resources
