## 📝 MODEL ANSWER & REFERENCE

### Sample Complete Template Library

**QUICK-STARTS (0-3 seconds):**
- "Well, let me think about that..."
- "That's an interesting question..."
- "To be honest, I'd say..."
- "Actually, I believe..."
- "I've never thought about that, but..."

**PART 1 FLUENCY:**
- "I [enlove/like/don't really care for] [topic] because..."
- "Generally speaking, I..."
- "It depends on [situation], but usually..."
- "To be honest, I don't [have much experience/know much] about..."

**PART 2 STRUCTURE:**
- Keywords: Subject + Detail + Feeling + Extension
- Opening: "I'd like to talk about..."
- Middle: "What's interesting about..." / "What really stands out..."
- Closing: "And that's why [topic] is [important/meaningful] to me."

**PART 3 PATTERNS:**

| Question Type | Formula |
|---------------|---------|
| Should | Position → Main reason → Example → Extension |
| Effect | Main effect → Additional effect → Example |
| Compare | Similarity → Difference → Why |
| Future | Prediction → Reason → Alternative |

**CONNECTORS:**
- Adding: "and," "also," "besides," "moreover"
- Contrasting: "but," "however," "although," "on the other hand"
- Example: "for instance," "such as," "take... for example"
- Result: "so," "therefore," "as a result"
- Conclusion: "overall," "in conclusion," "to sum up"

## 💡 COACH'S NOTES

### 🎓 Teaching Points

**1. Less is More**
> The best reference sheet is SIMPLE. If you can't scan it in 5 seconds, it's too complicated. One page is enough – don't overcomplicate.

**2. Personalization is Key**
Your template library should reflect YOUR speaking style:
- Use phrases YOU actually say
- Include YOUR universal examples
- Reflect YOUR strengths
- Address YOUR weaknesses

**3. This is NOT for Memorization**
The template library is a REMINDER, not a script. You should already know everything on it. It's just there to:
- Calm your nerves
- Provide structure under pressure
- Ensure consistency

### 🔧 Common Pitfalls & Fixes

| Pitfall | Coach's Fix |
|---------|-------------|
| **Making it too long** | Force yourself to fit on ONE page. If it doesn't fit, cut. |
| **Including too many options** | Pick 3-5 best phrases per category, not 15 |
| **Not practicing with it** | Use this library in your next mock exam. Make sure it works. |
| **Copying someone else's** | YOUR library = YOUR style. Customize everything. |
| **Never looking at it again** | Review this library weekly. Make it second nature. |

### 🎯 Pro Tips

1. **Make it visual** – use boxes, arrows, symbols for quick scanning
2. **Color-code** – different colors for different sections
3. **Phone wallpaper** – screenshot and set as lock screen for constant review
4. **Print multiple copies** – one in your bag, one at home, one备用
5. **Update as needed** – this is a living document, not fixed forever
