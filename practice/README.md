# Practice Materials

This directory contains practice exercises, mock tests, and skill-building activities organized by IELTS component.

## Structure

- `reading/` - Reading comprehension practice materials
- `writing/` - Writing task practice and sample responses  
- `grammar/` - Grammar exercises and language improvement drills

## Usage

Practice materials should be used regularly to develop and maintain IELTS skills:

1. **Daily Practice** - Complete short exercises daily
2. **Weekly Tests** - Take full practice tests weekly
3. **Skill Focus** - Target specific weaknesses with targeted exercises
4. **Progress Tracking** - Compare performance over time using progress tracking tools

## File Naming Convention

- Use date format: `YYYY-MM-DD-skill-topic.md`
- Example: `2024-01-15-reading-true-false.md`
- For test files: `YYYY-MM-DD-full-test.md`

## Best Practices

- Practice without time limits initially, then timed conditions
- Review answers and understand mistakes
- Keep track of progress and improvement areas
- Use reference materials to understand correct approaches
