# Task 9: Learn 5 Alternatives to "Very"

## 📚 PRE-STUDY [5 minutes]

### Before You Begin
**Why stop using "very"?**

"Very" is the **most overused word** in English learner speech:
- It's lazy vocabulary
- It limits your expression
- It signals limited language range to examiners
- There are ALWAYS better alternatives

**The "Very" Problem:**

| Instead of... | Say this... | Impact |
|---------------|-------------|--------|
| very good | excellent, outstanding | Stronger, more precise |
| very bad | terrible, awful | More emotional impact |
| very big | enormous, massive | Vivid imagery |
| very small | tiny, miniature | Precise description |

### Key Concepts

| Word Replacement | "Very" + Word | → | Better Alternative |
|------------------|---------------|---|-------------------|
| **Degree** | very big | → | huge, enormous, massive |
| **Quality** | very good | → | excellent, outstanding, superb |
| **Intensity** | very bad | → | terrible, awful, dreadful |
| **Amount** | very few | → | scarce, rare, minimal |
| **Speed** | very fast | → | rapid, swift, quick |

### The 5 Words You'll Learn Today

| Word | Meaning | When to Use |
|------|---------|-------------|
| **Incredibly** | Unusually, to an extreme degree | Strong emphasis, positive things |
| **Remarkably** | Noticeably, worth talking about | Surprising facts, achievements |
| **Fairly** | Moderately, to some extent | Neutral middle ground |
| **Rather** | Quite, more than expected | Polite understatement |
| **Somewhat** | Slightly, to some degree | Mild statements, uncertainty |

| very good | incredibly good | Stronger emphasis |
| very big | remarkably big | Adds surprise element |

| very tired | somewhat tired | Softer, less absolute |

### Usage Context Guide

**INCREDIBLY** - High intensity, positive or negative
- incredibly beautiful / incredibly difficult / incredibly kind
- Use when you want to show strong emotion

**REMARKABLY** - Surprise, worth noting
- remarkably similar / remarkably quick / remarkably calm
- Use when something is notable or unexpected

**FAIRLY** - Middle position, neutral
- fairly common / fairly simple / fairly well
- Use when something is average or moderate

**RATHER** - British preference, polite emphasis
- rather good / rather cold / rather disappointing
- Use to sound more sophisticated (especially in British contexts)

**SOMEWHAT** - Mild, uncertain, softening
- somewhat unusual / somewhat confusing / somewhat interested
- Use when you're not completely sure or want to be gentle


## 💡 COACH'S NOTES

### 🎓 Teaching Points

**1. Precision = Higher Band Score**
> IELTS examiners are trained to notice vocabulary range.
> "Very good" = Band 5 vocabulary level
> "Incredibly good" or "Outstanding" = Band 7+ vocabulary level
> Every time you replace "very," you're earning points.

**2. British vs. American Preference**
- **"Rather"** is more common in British English (IELTS is British)
- **"Pretty"** is the American equivalent of "rather"
- Both are acceptable, but "rather" sounds more formal

**3. Context Matters**
You wouldn't say "incredibly small" for something tiny.
Better: "minuscule" or "tiny"
Sometimes a specific word beats a degree word.

**Lesson:** These 5 words are tools, not rules. Think about meaning first.

### 🔧 Common Pitfalls & Fixes

| Pitfall | Problem | Coach's Fix |
|---------|---------|-------------|
| **Overusing one replacement** | "Incredibly" every time | Rotate all 5 words |
| **Using wrong intensity** | "Incredibly" for mild things | Match intensity to meaning |
| **Forgetting to use them** | Back to "very" out of habit | Practice until automatic |
| **Using in wrong context** | "Somewhat beautiful" | "Somewhat" = mild; beautiful = strong |
| **Sounding unnatural** | Forced word choice | Use these until they feel natural |

### 🎯 Pro Tips

1. **Set a phone reminder** - "Replace 'very'" 3x daily
2. **Keep a "very" journal** - note when you catch yourself using it
3. **Use one word per day** - focus on mastering each individually
4. **Record yourself** - check if you're actually using the new words
5. **Read your sentences aloud** - muscle memory helps retention
