# Task 13: "One Thing That..." Sentence Pattern

## 📚 PRE-STUDY [5 minutes]

### Before You Begin
**Why This Pattern is Powerful**

"One thing that has significantly changed my life is..." is a **high-value sentence structure** because:
- It shows grammatical range (relative clauses)
- It sounds sophisticated and natural
- It works for almost ANY topic
- It buys you thinking time (long introduction)
- It impresses IELTS examiners

**The Pattern Structure:**

```
"One thing that has [past participle] [noun] is [explanation]..."
```

### Key Concepts

| Element | Function | Examples |
|---------|----------|----------|
| **"One thing that"** | Introduces a specific point | One thing that... |
| **has + past participle** | Shows connection to present | has changed / has improved / has influenced |
| **your noun phrase** | What you're discussing | my life / my thinking / my English |
| **"is"** | Begins explanation | is... |
| **the explanation** | Your main point | daily practice / this app / meeting you |

### Variations of This Pattern

| Pattern | When to Use |
|---------|-------------|
| One thing that has changed... | Personal transformations |
| One thing that has improved... | Positive developments |
| One thing that has influenced... | Important factors |
| One thing that has helped... | Support, resources |
| One thing that has taught me... | Lessons learned |

## 📝 MODEL ANSWER & REFERENCE

### Sample Sentences Using the Pattern

**Example 1 - Changed (Personal)**
"One thing that has significantly changed my life is learning to manage my time better. I used to procrastinate a lot, but now I use a planner and prioritize tasks, which has made me much more productive and less stressed."

**Example 2 - Improved (Skill)**
"One thing that has greatly improved my English is watching TED Talks without subtitles. At first it was difficult, but now I can understand much more, and I've learned many natural expressions that native speakers actually use."

**Example 3 - Influenced (Person/Event)**
"One thing that has deeply influenced my career choice was meeting my current manager. She showed me what's possible in this field, and her mentorship gave me the confidence to pursue leadership roles."

**Example 4 - Helped (Resource/Tool)**
"One thing that has really helped me prepare for IELTS is using flashcard apps for vocabulary. I can review anytime, anywhere—on the bus, waiting in line—and this consistency has made a big difference in my word retention."

## 💡 COACH'S NOTES

### 🎓 Teaching Points

**1. Why This Pattern Impresses Examiners**
> The "One thing that..." structure demonstrates several grammar points simultaneously:
> - Relative clauses ("that has changed...")
> - Present perfect tense ("has changed")
> - Complex sentence structure
> - Sophisticated vocabulary (when used well)
>
> One sentence can showcase multiple Band 7+ features.

**2. The "Thinking Time" Benefit**
This pattern is long! "One thing that has significantly changed my life is..." = 11 words before you even state your point.
This buys you precious seconds to think while you're speaking.
Brilliant for when your mind goes blank.

**3. Adaptability**
The pattern works because:
- It's flexible (change/improve/influence/help/teach)
- It's personal (your life/your English/your career)
- It's expandable (add as much detail as you want)

**Make this pattern YOURS. Use it often.**

### 🔧 Common Pitfalls & Fixes

| Pitfall | Problem | Coach's Fix |
|---------|---------|-------------|
| **Forgetting "that"** | Grammatically incomplete | "One thing **that** has..." is essential |
| **Wrong tense** | "have changed" instead of "has" | Singular "thing" = "has" |
| **Not completing the thought** | "...is" and then nothing | Prepare the full explanation |
| **Overusing the same verb** | "changed" every time | Rotate: improved, influenced, helped, taught |
| **Too long, rambling** | Loses the listener | Keep explanation focused |
| **Not practicing aloud** | Can't use it under pressure | Say each sentence 10+ times |

### 🎯 Pro Tips

1. **Memorize 3 variations** - have ready examples for common topics
2. **Use it as a starter** - begin Part 2 responses with this pattern
3. **Record yourself** - make sure it sounds natural, not rehearsed
4. **Add emphasis words** - "significantly," "greatly," "really"
5. **Practice the pronunciation** - "One thing that..." should flow smoothly
