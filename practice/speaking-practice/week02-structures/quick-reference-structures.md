## 🔗 Relative Clause Patterns

### Basic vs. Advanced

| Band 5-6 | Band 7+ |
|----------|---------|
| "My friend **who is a doctor** lives here." | "My friend, **a doctor by profession**, lives here." |
| "The book **that I read** was good." | "The book **I recently finished** was excellent." |
| "I like places **which are quiet**." | "I prefer places **offering peace and tranquility**." |

### Quick Templates

```markdown
**People**: "My [relation], [who/that] [action], is [description]."
**Things**: "The [thing] [which/that] I [verb] is [opinion]."
**Places**: "I enjoy [place], [where] [activity happens]."
```

**Example**:
> "My teacher, **who has been teaching for 10 years**, gives great advice."

## 🎯 Conditional Patterns

### Zero & First Conditionals (Real/Habitual)

| Type | Structure | Example |
|------|-----------|---------|
| **Zero** | If + present, present | "If **I practice**, **I improve**." |
| **First** | If + present, will + verb | "If **I study hard**, **I will get** Band 7." |

### Second Conditional (Hypothetical - Band 7+)

```markdown
"If + past simple, would + verb"
```

| Question | Band 6 Response | Band 7+ Response |
|-----------|----------------|-----------------|
| "If you had more time..." | "If I **have** more time, I **will**..." | "If I **had** more time, I **would**..." |
| "Where would you go?" | "I **will** go to..." | "I **would** choose to visit..." |

**Quick Template**:
```
"If I [past situation], I would [hypothetical action]. For example, if I had more free time, I would dedicate it to learning new skills."
```

## 🎲 Modal Verbs for Speculation

### Common Patterns in Part 3

| Modal | Use | Example |
|-------|-----|---------|
| **might** | Possibility (30-50%) | "People **might** choose to work remotely." |
| **could** | Possibility (40-60%) | "This **could** lead to better work-life balance." |
| **would** | Hypothetical result | "It **would** allow more flexibility." |
| **may** | Possibility (50-50%) | "This **may** become more common in the future." |
| **should** | Recommendation | "Companies **should** offer flexible options." |

### Band-Boosting Formula

```markdown
Simple: "I think X will happen."
Advanced: "X **might** lead to Y, which **could** result in Z."
```

**Example**:
> "Remote work **might** become more popular, which **could** reduce traffic in cities."

## 🔄 Part 2 Structure Templates

### 1-Minute Note Template

```markdown
Topic: [What to describe]
    ↓
Point 1: What it is
Point 2: When/Where/Who
Point 3: Why important
Point 4: Feelings/Opinion
```

### Quick Extension Formula

Running out of things to say? Add:

| Detail Type | Formula |
|-------------|---------|
| **Description** | "It looks like... / It's made of..." |
| **Comparison** | "Unlike X, Y is..." |
| **Personal Story** | "I remember when I..." |
| **Opinion** | "What I love about it is..." |
| **Future** | "In the future, I hope to..." |

## 🌟 Part 3 Discussion Language

### Stating Opinions (Band 7+)

| Simple | Advanced |
|--------|----------|
| "I think..." | "**From my perspective**..." / "**As I see it**..." |
| "I agree..." | "I **couldn't agree more**." / "I **tend to agree**..." |
| "In my opinion..." | "**Ideally speaking**..." / "**Generally speaking**..." |

### Explaining Reasons

```markdown
"The main reason is that..."
"This is primarily because..."
"There are several factors involved. Firstly,..."
```

### Adding Examples

```markdown
"For instance,... / For example,..."
"A good case in point is..."
"Take... for example."
"To illustrate..."
```

## 📊 Band 7+ Vocabulary Upgrades

### Common Replacements

| Basic | Band 7+ |
|-------|---------|
| **good** | beneficial, advantageous, valuable |
| **bad** | detrimental, harmful, problematic |
| **important** | crucial, significant, essential |
| **big** | substantial, considerable, notable |
| **many** | numerous, various, a multitude of |
| **think** | believe, consider, maintain |
| **show** | demonstrate, illustrate, indicate |
| **get** | acquire, obtain, achieve |

### Collocations for Band 7+

- **have a significant impact on**
- **play a crucial role in**
- **take into consideration**
- **come to a conclusion**
- **make a decision**
