# Speaking Practice: 100 Tasks Checklist

> **Quick printable checklist** for tracking IELTS Speaking practice progress.
>
> For detailed task instructions, see [`口语练习材料/`](../口语练习材料/) directory.

---

## Usage

- **Print** this checklist and mark progress manually
- **Use as reference** when planning weekly practice
- **For detailed guides**, see individual task files in `口语练习材料/`

## Task Structure

| Week | Focus | Tasks |
|------|-------|-------|
| **Week 1** | Fluency Foundation | Tasks 1-25 |
| **Week 2** | Sentence Expansion | Tasks 26-50 |
| **Week 3** | Pronunciation + Mocks | Tasks 51-75 |
| **Week 4** | Final Sprint | Tasks 76-100 |

---

## 第一周：打地基（流利度启动）

- [ ] 1. 下载 ELSA Speak，完成发音诊断测试，记录初始分数
- [ ] 2. 列出10个雅思高频话题：家乡/工作/科技/旅行/教育/健康/环境/娱乐/朋友/未来计划
- [ ] 3. 为每个话题准备3个关键点框架，只记结构不翻译
- [ ] 4. 录音①：60秒自我介绍，不管语法，只求不停顿
- [ ] 5. 回放录音①，统计停顿超过2秒的次数，记录数字作为基线
- [ ] 6. 听 BBC 6 Minute English 一集，标记3个你想用的句子
- [ ] 7. 跟读Day1：选60秒片段跟读3遍，专注节奏，不纠发音
- [ ] 8. 完成"家乡"话题：连续说1分钟，录音
- [ ] 9. 学5个替代very的词：incredibly / remarkably / fairly / rather / somewhat
- [ ] 10. 用这5个词各造一个句子，开口说出来
- [ ] 11. 跟读Day2：同一片段，这次模仿语调起伏
- [ ] 12. 完成"工作/学习"话题1分钟录音
- [ ] 13. 用 "One thing that has significantly changed my life is..." 造3个句子并说出来
- [ ] 14. ELSA专项练习：练你最弱的5个音标，每个练10遍
- [ ] 15. 跟读Day3：换新片段，先完整听一遍再跟读
- [ ] 16. 完成"科技"话题1分钟录音
- [ ] 17. 学5个连接词：Furthermore / In contrast / What's more / As a result / That said
- [ ] 18. 用这5个连接词重新讲一遍"家乡"话题
- [ ] 19. 跟读Day4：注意连读，如 turn it on 读成 tur-ni-ton
- [ ] 20. ELSA专项：-ed结尾发音 /t/ /d/ /ɪd/，各练5个词
- [ ] 21. 完成"旅行"话题录音，强制用上至少2个连接词
- [ ] 22. 回顾本周4段录音，对比停顿次数是否减少，写下观察
- [ ] 23. 跟读Day5：只跟读，不录音，纯感受节奏
- [ ] 24. 找语伴或用AI做5分钟随机问答练习
- [ ] 25. 第一周总复盘：写1个本周最大进步 + 1个下周重点改进方向

---

## 第二周：句式扩展

- [ ] 26. 学定语从句模板：...which/that + verb...，造5个例句并说出来
- [ ] 27. 学条件句模板：If I had the chance, I would...，造3个例句
- [ ] 28. 完成"教育"话题录音，每个观点后强制加一句具体举例
- [ ] 29. ELSA专项：-ing结尾词语连读时不要吞音，练10个词
- [ ] 30. 跟读Day6：选TED-Ed短片，速度调到0.85x
- [ ] 31. 学情态搭配：tend to / be likely to / be inclined to，各造2句
- [ ] 32. 完成"健康"话题录音，至少用1个定语从句
- [ ] 33. 回放录音，检查有无简单句堆叠，标出3处可改进的地方
- [ ] 34. ELSA专项：th音 /θ/（think）和 /ð/（this），各练10个词
- [ ] 35. 跟读Day7：速度恢复正常，跟读时同步录音
- [ ] 36. 学对比句式：While some people believe... others argue that...
- [ ] 37. 完成"环境"话题录音，用对比句式表达两种观点
- [ ] 38. 找10个高频形容词，各配一个更精准的同义词（big→substantial，good→remarkable）
- [ ] 39. ELSA专项：单词重音位置，练10个多音节词（important / economy / technology）
- [ ] 40. 跟读Day8：只跟读第一句，反复10次直到完全自然
- [ ] 41. 录音：自由谈"你最喜欢的电影或书"，不准备直接说2分钟
- [ ] 42. 回放上条录音，标出哪些地方用了复杂句式，哪些还是简单句堆叠
- [ ] 43. 学Part 2答题结构：Topic句 + Background + Detail + Personal feeling
- [ ] 44. 完成"娱乐"话题录音，严格按Part 2四段结构组织
- [ ] 45. ELSA专项：句子语调训练，问句vs陈述句的语调区别
- [ ] 46. 跟读Day9：选正常语速BBC片段，不降速，跟上即可
- [ ] 47. 把本周学的3个句式模板写成15个可直接套用的句子，存入手机备忘录
- [ ] 48. 完成"朋友"话题录音，计时2分钟不间断
- [ ] 49. 第二周复盘：从本周录音中找出进步最明显的一个句式
- [ ] 50. 模拟Part 1：随机抽3个话题，每个答2-3句，完整录音

---

## 第三周：发音精修 + 模拟考

- [ ] 51. 完整模拟考①：Part1(4分钟)+Part2(2分钟)+Part3(4分钟)，全程录音不暂停
- [ ] 52. 回放模拟录音，用流利度/发音/词汇三个维度各打1-10分，记录分数
- [ ] 53. ELSA专项：找出发音最差的5个词，每个练20遍
- [ ] 54. 跟读Day10：专门选发音清晰的英音或美音主播片段
- [ ] 55. 学弱读规则：a/an/the/and/of/for 在句中读弱化音，不重读
- [ ] 56. 用弱读规则重新朗读5个句子，录音前后对比
- [ ] 57. 完成"未来计划"话题，用Part 2结构 + 至少3个复杂句式
- [ ] 58. ELSA专项：/æ/（cat）vs /e/（bed）vs /ʌ/（cup）三个元音区分训练
- [ ] 59. 跟读Day11：模仿语调起伏，刻意夸张，求感受不求自然
- [ ] 60. 练习Filler替代：把 um/uh 全换成 Well... / Actually... / That's a good question...
- [ ] 61. 录音：Part 3延伸问答练习，每道题答至少3句，全程不用um/uh
- [ ] 62. ELSA专项：/r/ 和 /l/ 区分训练（中文母语者高频问题）
- [ ] 63. 跟读Day12：只跟发音不管内容，盯着音标练
- [ ] 64. 完整模拟考②：计时，不暂停，模拟真实考场氛围
- [ ] 65. 对比模拟①和②的自评分，写下分数变化和原因分析
- [ ] 66. 学"买时间"技巧：That's an interesting question, let me think... / From my perspective...
- [ ] 67. ELSA专项：连读规则，辅音+元音连读（pick it up → pi-ki-tup）
- [ ] 68. 跟读Day13：选对话片段而非独白，练习对话语调
- [ ] 69. 完成任意2个话题，全程不允许中文思维，用英文想再开口说
- [ ] 70. 整理本周错误发音清单，目标记录不少于15个词
- [ ] 71. ELSA专项：句子重音训练，每句话只突出1-2个关键词
- [ ] 72. 跟读Day14：正常语速跟读，跟完后立刻回答一个相关问题
- [ ] 73. 录音：描述一张你喜欢的照片，2分钟不间断
- [ ] 74. 第三周复盘：发音哪个维度进步最大？具体写出来
- [ ] 75. 给自己写一段英文本周总结，不少于5句话，说出来录音

---

## 第四周：冲刺与稳定发挥

- [ ] 76. 完整模拟考③：模拟真实考场，坐直、计时、不重来
- [ ] 77. ELSA冲刺：把三周积累的错误清单里的词全部重新练一遍
- [ ] 78. 准备10个"万能例子"，每个可用于多个话题（如：智能手机改变生活/疫情影响/朋友的故事）
- [ ] 79. 跟读Day15：把最喜欢的一段练到脱口而出，不看文字
- [ ] 80. 录音：用万能例子分别回答3个不同话题，验证通用性
- [ ] 81. 练习Part 2准备节奏：计时60秒只写3个关键词，然后说2分钟
- [ ] 82. ELSA专项：最不确定的5个词，查音标后再集中练
- [ ] 83. Part 3专项：抽4-5道抽象问题（Should the government...? / What are the effects of...?），每道答3句以上
- [ ] 84. 录音：用最自然的语气讲一件最近发生的真实小事，3分钟
- [ ] 85. 回放上条录音：检查语调是否单调，标出应该有起伏的地方
- [ ] 86. 跟读Day16：找情感丰富的片段，模仿情绪而不只是发音
- [ ] 87. 练习快速开口：听到问题后3秒内必须开口，哪怕从 Well... 开始
- [ ] 88. 完整模拟考④：找朋友或用AI扮演陌生考官，增加陌生感压力
- [ ] 89. 整理最终模板库：流利度模板+句式模板+万能例子，打印成一页纸
- [ ] 90. ELSA专项：模拟完整对话场景，不只练孤立的单词
- [ ] 91. 跟读Day17：正常语速完整跟读，录音留存作为进步对比素材
- [ ] 92. 重新录一段自我介绍，和第一周第4条的录音做对比，感受差距
- [ ] 93. 练习考场状态管理：坐姿端正，有意识控制语速在120-150词/分钟
- [ ] 94. ELSA最终评估：做一次完整发音测试，记录最终分数，与初始分数对比
- [ ] 95. 写备考总结：哪3件训练最有效？哪1件回头看在浪费时间？
- [ ] 96. 完整模拟考⑤（最后一次）：把它当真正的考试，全力以赴
- [ ] 97. 回放最后一次模拟，只找亮点不找问题，建立考前信心
- [ ] 98. 考前48小时：不做新内容，只回顾模板库和万能例子
- [ ] 99. 考试当天早上：做一次15分钟跟读热身，激活口腔肌肉
- [ ] 100. 完成全部100条！用英语给自己说一句鼓励的话，录音留存
