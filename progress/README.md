# Progress Tracking

This directory tracks your IELTS preparation progress, including performance metrics, improvement areas, and goal achievements.

## Structure

- `weekly-reports/` - Detailed weekly progress summaries and analysis
- `stats/` - Statistical data and performance metrics
- `goals/` - Short and long-term IELTS preparation goals

## Usage

Progress tracking should be systematic and regular:

1. **Weekly Updates** - Create detailed weekly progress reports
2. **Metrics Collection** - Track scores, time, and accuracy metrics
3. **Goal Setting** - Set specific, measurable goals for improvement
4. **Analysis** - Review progress to identify trends and areas for improvement

## File Naming Convention

- Weekly reports: `YYYY-MM-DD-weekly-report.md`
- Stats: `YYYY-MM-DD-skill-stats.md`
- Goals: `YYYY-MM-DD-goals.md`

## Key Metrics to Track

- Practice scores and band estimates
- Time management performance
- Accuracy by question type
- Vocabulary and grammar improvement
- Consistency of performance

## Analysis Framework

Use regular analysis to:
- Identify strengths and weaknesses
- Track improvement over time
- Adjust study strategies
- Set realistic timelines for target band scores
